# 김희주의 유니버스

HTML, CSS, JavaScript 기초 과제의 필수 요소를 유지하면서 반응형 UI와 브라우저 API를 확장 구현한 개인 웹사이트입니다.

## 실행 방법

별도의 빌드 과정은 없습니다. VS Code Live Server처럼 로컬 웹 서버를 실행한 뒤 `html/index.html`을 열면 됩니다. ES Module과 Web Crypto API를 사용하므로 파일을 직접 여는 방식보다 `localhost` 환경을 권장합니다.

## 주요 구현

- 프로필, 수업 시간표, 휴일 일과, 여행 앨범, 회원가입 등 과제 필수 페이지
- 라이트/다크 테마 전환과 사용자 선택 저장
- 첫 방문 타이핑 인트로와 레이아웃 조립 애니메이션
- 스크롤 진입 시 순차적으로 나타나는 콘텐츠 애니메이션
- 반응형 레이아웃과 모바일 햄버거 메뉴
- 회원가입, 로그인, 로그아웃, 마이페이지
- 사용자별 주말 일정 추가·수정·삭제·전체 삭제
- 현재 위치 확인, 도시 검색 및 날씨 조회
- 여행 사진 라이트박스와 배경 음악 자동 재생
- GitHub·PDF 포트폴리오 바로가기와 프로젝트 문제 해결 기록
- 개발자로서의 3문 3답과 공통 CONTACT 푸터
- 과제 필수 `prompt`·`alert` 방식과 커스텀 `dialog` 방식을 선택할 수 있는 간편 도구

## CSS 구조

- `style.css`: 디자인 토큰, reset, 공통 타이포그래피와 레이아웃, header/footer
- `components.css`: 폼 컨트롤, 버튼, 시간표 등 재사용 컴포넌트
- `pages.css`: 프로필, 휴일, 여행, 인증, 마이페이지, 플래너
- `home.css`: 인덱스 대시보드와 utility 전용 스타일
- `motion.css`: 홈 인트로, 스크롤 reveal, keyframes
- `responsive.css`: breakpoint와 reduced-motion 대응

모든 페이지는 과제 요구사항에 따라 `style.css`를 먼저 불러오며, `responsive.css`는 cascade를 유지하기 위해 마지막에 불러옵니다.

## 인증과 저장 방식

가입한 회원 정보는 `localStorage`에 저장됩니다. 로그인 상태는 기본적으로 현재 탭을 닫으면 종료되며, 로그인 화면에서 `로그인 상태 유지`를 선택한 경우에만 브라우저를 다시 열어도 유지됩니다. 비밀번호 원문 대신 브라우저의 Web Crypto API로 생성한 해시를 저장하며, 휴일 페이지의 개인 일정은 사용자 아이디별 키로 분리됩니다.

이 인증은 프론트엔드 학습용 데모입니다. 서버와 데이터베이스가 없으므로 실제 서비스의 보안·인증 수단으로 사용할 수 없습니다.

## 접근성 및 UX

- 키보드로 메뉴, 폼, 라이트박스 조작 가능
- 현재 페이지와 폼 오류 상태를 보조 기술에 전달
- `prefers-reduced-motion` 환경에서는 장식 애니메이션 최소화
- 인트로는 같은 탭에서 최초 1회만 재생하며 건너뛰기 제공
- 작은 화면에서는 내비게이션과 폼, 카드 레이아웃을 단일 열로 전환
