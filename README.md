# 김희주의 유니버스

HTML, CSS, JavaScript 과제의 필수 기능을 바탕으로 개인 콘텐츠와 여러 웹 기능을 확장한 포트폴리오형 웹사이트입니다.

## 실행 방법

별도의 설치나 빌드 과정은 없습니다. VS Code Live Server와 같은 로컬 웹 서버를 실행한 뒤 `html/index.html`을 열어주세요.

ES Module과 Web Crypto API를 사용하므로 HTML 파일을 직접 실행하는 방식보다 `localhost` 환경을 권장합니다.

## 과제 필수 구현

### HTML

- 지정된 제목과 환영 문구가 있는 메인 페이지
- 필수 태그를 활용한 휴일 일과와 프로필 목록
- `thead`, `tbody`, 셀 병합을 활용한 수업 시간표
- `fieldset`, 입력 요소, Submit·Reset을 포함한 GET 방식 회원가입 폼과 결과 페이지
- `audio`, `img`, `video`, `source`를 활용한 여행 앨범
- 페이지 바로가기와 `nav`, `main`, `aside`로 구성한 개인 포털

### CSS

- Google Fonts, 색상, 줄 간격, 제목과 링크 스타일을 적용한 공통 CSS
- 컨테이너와 박스 모델을 활용한 중앙 정렬 및 여행 카드·시간표·회원가입 폼 디자인
- Flexbox로 구성한 바로가기와 메인·사이드바 배치
- Grid를 활용한 3열 여행 카드와 786px 이하의 단일 열 반응형 레이아웃
- 내비게이션·버튼 전환, 여행 카드 hover, 메인 타이틀 진입 애니메이션

### JavaScript

- `prompt`, `alert`, 반복문을 활용한 Up-Down 게임과 성적 계산기
- 객체 배열과 반복문을 활용한 가방 속 물품 출력
- 도시 선택 `change` 이벤트와 DOM 조작을 통한 도시명·위도·경도 표시
- `fetch`와 `async/await`를 활용한 Open-Meteo 온도·습도 조회 및 로딩 상태 표시
- `weatherAPI.js`와 `realtimeInfo.js`로 분리한 데이터 요청·화면 처리 모듈

## 추가 구현

- 라이트·다크 테마 전환 및 사용자 설정 저장
- 최초 방문 시 한 글자씩 나타나는 타이핑 인트로와 순차적인 스크롤 진입 애니메이션
- 회원가입, 로그인, 로그아웃, 마이페이지 및 로그인 상태에 따라 바뀌는 내비게이션
- 입력값 검증, 비밀번호 표시 전환, 이메일 직접 입력, 자기소개 글자 수 확인을 포함한 회원가입 폼
- 비밀번호를 salt와 SHA-256 해시로 저장하고 로그인 유지 여부에 따라 세션을 관리하는 인증 기능
- 브라우저 저장소를 활용한 사용자별 휴일 일정 추가·수정·삭제·정렬
- 위치 권한 상태 처리와 역지오코딩을 활용한 현재 지역 표시
- 도시 검색, 로딩 상태, 검색 간격 및 캐시를 적용한 날씨 조회
- 화면 높이에 맞을 때만 따라오는 Utility 영역
- 786px 이하에서 햄버거 내비게이션 제공
- 스크롤 위치에 따라 나타나고 푸터와 겹치지 않는 상단 이동 버튼
- 키보드 방향키를 지원하는 여행 사진 라이트박스
- 여행 배경 음악 자동 재생과 갑작스러운 소음을 줄이기 위한 기본 음량 25% 설정
- 과제용 Alert 방식과 UI를 개선한 Dialog 방식의 간편 도구
- `prefers-reduced-motion`을 반영한 애니메이션 최소화
- 역할에 따라 분리한 CSS와 기능별 JavaScript 모듈 구성
- GitHub 및 [포트폴리오 PDF](./documents/portfolio.pdf) 연결

## 폴더 구조

```text
html/       페이지
css/        공통·컴포넌트·페이지별 스타일
script/     기능별 JavaScript
media/      이미지·오디오·비디오
documents/  포트폴리오 문서
```

> 회원가입과 로그인 기능은 프론트엔드 학습용으로 구현했습니다. 사용자 정보는 브라우저 저장소에 보관되며 실제 서비스의 인증 수단으로 사용할 수 없습니다.
